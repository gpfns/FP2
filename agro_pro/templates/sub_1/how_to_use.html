How To Use


SmartAgroPro Android Application provides the following Features :
  1. Optimal Crop Prediction
  2. Get Current Climatic Conditions
  3. Weather Forecast for upcoming days
  4. Current Market Prices of various Commodities
  5. Minimum Support Price Details
  6. Region Specific Crop Enhancement Suggestions

To Find out the Optimal Crop, First Click on Get Started and Select Crop Prediction. Here you can choose Basic Crop Prediction or Advanced Crop Prediction. The Advanced Crop Prediction provides accurate results and is a Recommended and Recognized model. The Basic Crop Prediction is simple, fast, easy to use which requires only providing User location.

Current Weather Data or Future Weather Forecast can be found by using Weather Forecast Button.

Current Prices of Various Commodities or Minimum Support Prices the Government provides can be known by selecting Market Prices button.
